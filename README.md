گاراژ محتوا
این مخزن شامل کدهای  HTML، CSS و JavaScript مربوط به گاراژ محتوا است؛ صفحه وبی که به عنوان پورتفولیوی شخصی من برای معرفی پروژه‌های محتوایی، پژوهش‌ها و تجربه‌های حرفه‌ای طراحی شده است. هدف این پروژه، گردآوری مجموعه‌ای از محتواهای مستقل در یک فضای ساده، سریع و در دسترس است.
درباره این پروژه
گاراژ محتوا آرشیوی از پروژه‌هایی است که هرکدام از یک مسئله، یک سؤال یا یک کنجکاوی واقعی آغاز شده‌اند. این پروژه با تمرکز بر سادگی، تجربه کاربری مناسب و دسترسی آسان به پروژه‌های محتوایی توسعه یافته و بستری برای معرفی فعالیت‌های من در حوزه تولید محتوا، پژوهش و استفاده کاربردی از هوش مصنوعی فراهم می‌کند.
